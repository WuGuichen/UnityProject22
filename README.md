## 介绍
* 该项目主要基于《设计模式与游戏完美开发》书中框架结构。游戏类型为第三人称ARPG游戏。
### 已有功能
1. 移动端和PC端的输入
2. UI管理
3. 敌人和玩家两个阵营的角色管理
4. 动画控制
5. 游戏资源管理
6. 游戏属性和角色属性管理
7. 等等
